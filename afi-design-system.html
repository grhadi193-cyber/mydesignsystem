توکن‌های طراحی

تمام مقادیر پایه سیستم از طریق CSS Custom Properties در :root تعریف شده‌اند. این توکن‌ها در تمام کامپوننت‌ها استفاده می‌شوند.

/* همه توکن‌ها در :root تعریف می‌شوند */ :root { /* رنگ‌های پس‌زمینه */ --bg-primary: #FAFAF7; --bg-secondary: #F0EDE6; --bg-tertiary: #E5E1D8; /* برند */ --navy: #1A56DB; --teal: #0891B2; --amber: #D97706; /* انیمیشن */ --transition-fast: 180ms cubic-bezier(0.4, 0, 0.2, 1); --transition-base: 250ms ease; }
